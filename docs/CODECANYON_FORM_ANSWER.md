## 1) Name & Description

### Name
**WarungWA - WhatsApp Order Management CRM for Small Businesses**

> Panjang < 100 karakter, tanpa HTML/emoji, sudah deskriptif.

### HTML Description
Gunakan deskripsi berikut (boleh plain text atau HTML sederhana):

```html
<h2>WarungWA - WhatsApp Order Management CRM</h2>
<p>WarungWA is a frontend-only web app for small businesses to manage products, customers, and order pipelines, then close deals faster through WhatsApp click-to-chat integration.</p>

<h3>Main Features</h3>
<ul>
  <li>Product catalog management (CRUD, categories, variants, active/inactive status)</li>
  <li>Customer CRM (tags, levels, notes, order history, duplicate WhatsApp detection)</li>
  <li>Order pipeline (New → Confirmed → Packed → Shipped → Completed)</li>
  <li>Kanban board with drag & drop and list view</li>
  <li>WhatsApp message templates with variables ({name}, {total}, {order_id}, {address})</li>
  <li>PDF invoice generation from browser</li>
  <li>Analytics dashboard and CSV/Excel export</li>
  <li>Multi-store support and role-based access (Owner/Admin/Staff)</li>
  <li>Backup/restore data (JSON export/import)</li>
  <li>Responsive UI and offline-first architecture using IndexedDB</li>
</ul>

<h3>Tech Stack</h3>
<p>React 19, Vite, TypeScript, Tailwind CSS, React Router, Zustand, Dexie.js, @dnd-kit, React Hook Form, Zod, @react-pdf/renderer, Recharts.</p>

<h3>Important Note</h3>
<p>This item uses WhatsApp click-to-chat (wa.me) and does not include official WhatsApp Business API integration.</p>
```

## 3) Files Upload Checklist

### Thumbnail
- **File:** `thumbnail-80x80.png`
- **Spec:** JPEG/PNG, **80x80 px**

### Inline Preview Image
- **File:** `preview-590x300.jpg` (atau `.png`)
- **Spec:** **590x300 px**, JPEG disarankan

### Main File(s)
- **Format:** ZIP
- **Isi:** Semua file untuk buyer (source code + docs), **tanpa preview images**
- **Contoh nama:** `warungwa-main-files.zip`

### Preview Screenshots
- **Format:** ZIP
- **Isi:** Gambar `.png/.jpg` + optional `descriptions.txt`
- **Contoh nama:** `warungwa-preview-screenshots.zip`

### Optional Live Preview
- **Format:** ZIP
- **Wajib:** ada `index.html` (case-sensitive) + assets pendukung
- **Contoh nama:** `warungwa-live-preview.zip`

### Optional Video Preview
- **Format:** MP4 (lebih disarankan daripada FLV)
- **Isi:** demo fitur utama 30–120 detik
- **Resolusi:** isi di field “Video Preview Resolution” (contoh: `1280x720`)

> Catatan ukuran: upload web max 300MB. Jika >300MB pakai FTP (hingga 3GB).
