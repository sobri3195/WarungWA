# Preview Screenshots

Folder ini berisi paket ZIP screenshot preview.

## Output generator
- `warungwa-preview-screenshots.zip`

## Sumber file
ZIP ini mengambil file dari folder root `preview-screenshots/` dengan include:
- `*.png`
- `*.jpg`
- `descriptions.txt`

## Cara generate
Jalankan dari root project:

```bash
./upload-checklist/build-upload-files.sh
```
