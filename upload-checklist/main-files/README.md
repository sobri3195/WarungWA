# Main Files

Folder ini berisi paket ZIP utama yang diunggah ke marketplace.

## Output generator
- `warungwa-main-files.zip`

## Isi ZIP utama
Paket ini dibuat dari tracked files Git, dengan pengecualian file preview/upload biner agar paket tetap bersih dan ringan.

## Cara generate
Jalankan dari root project:

```bash
./upload-checklist/build-upload-files.sh
```
